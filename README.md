sea-battle-mvc-pattern
Sea Battle MVC Pattern — это классическая настольная игра "Морской бой", реализованная на Java с использованием архитектурного паттерна MVC (Model–View–Controller) и библиотеки Swing для построения графического интерфейса. Пользователь играет против виртуального сервера (компьютера). Проект демонстрирует чистое разделение логики, представления и управления, а также включает многоуровневую систему.

Ключевые особенности
Архитектура MVC: Чёткое разделение на Model, Viewer и Controller.

Система уровней: Игра включает 3 последовательных уровня сложности.

Противник: Игра ведется против компьютера.

Управление кораблями: Расстановка и перетаскивание кораблей.

Звуковое сопровождение: Интеграция фоновой музыки (AudioPlayer).

Требования
Перед запуском убедитесь, что у вас установлено следующее ПО:

Java JDK 17 (или выше)

Atom (или любая другая IDE/Редактор кода)

Командная строка (CMD/PowerShell) для сборки и запуска

Установка и запуск
1. Скачайте или клонируйте проект
Перейдите в папку, где вы хотите разместить проект, и выполните:

Bash

git clone <URL вашего репозитория>
cd sea-battle-mvc-pattern
2. Откройте проект в редакторе кода
В вашей IDE или редакторе кода откройте папку проекта:

File → Open Folder → выберите папку sea-battle-mvc-pattern

3. Проверьте установку JDK 17
Введите в терминале команду:

Bash

java --version
Убедитесь, что вывод указывает на версию 17 или выше.

4. Сборка и запуск через командную строку
Для компиляции и запуска проекта выполните две последовательные команды из корневого каталога sea-battle-mvc-pattern:

1. Компиляция
Эта команда компилирует все файлы .java в папке src, используя кодировку UTF-8, и помещает скомпилированные классы в папку out.

Bash

javac -encoding UTF-8 -d out src\*.java
2. Запуск
Эта команда запускает скомпилированный код, указывая путь к классовым файлам (-cp out) и запуская главный класс (Main).

Bash

java -cp out Main
5. Запуск через IDE
Если вы используете IntelliJ IDEA или Eclipse:

Убедитесь, что в настройках проекта (Project Structure) выбран JDK 17.

Найдите класс Main.java и запустите его.
